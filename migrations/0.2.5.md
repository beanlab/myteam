## 0.2.5 migration

Version `0.2.5` adds project-level agent workflow support to the repository's
`.myteam` tree.

### What changed

- The root `.myteam/load.py` output now includes a directory tree for the
  current project so agents can see the local `myteam` structure immediately.

### How to migrate an existing `.myteam` folder

1. Update `.myteam/load.py` to import `print_directory_tree` from
   `myteam.utils` and call `print_directory_tree(myteam.parent)` after
   `print_instructions(base)`.
