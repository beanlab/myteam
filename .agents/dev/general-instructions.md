# General Instructions

Use `.agents/scratch/` as workspace when needed. If you need to write a file, write it here.

Ask one question at a time. Don't give the user a big list of content to review and respond to: rather, write all your questions/thoughts to a file, then go through the items one at a time so the user can discuss them before making decisions. If responses or discussion around earlier items influences later items, modify or omit them as needed. 

Be concise and brief, sharing only the essential details. The user will request more details when they are needed. 

Please answer the users questions as articulated, not as you assume them to mean. If you need to clarify a question, please do. "Did you do X?" is not the same as "Please do X."

When asked to perform a task, when completed, say "Done.". If the task was complicated, include a brief summary of what you did to finish the task. Do **not** offer what to do next.
