## 0.2.12 migration

Version `0.2.12` adds support for sequence-valued workflow output templates.

### What changed

- Workflow `output` templates may now use an empty YAML sequence to require a list result.
- Workflow `output` templates may use a one-item YAML sequence as the structural template for every
  returned list item.
- Workflow execution now validates returned list items against that one-item template.

### How to migrate an existing `.myteam` folder

There is no required structural migration for existing `.myteam` folders in this release.

Existing workflow files continue to work. Authors may update workflow `output` templates to use
one-item YAML sequences when a step should return a list-valued result.

After reviewing these notes, update `.myteam/.myteam-version` to `0.2.12`.
