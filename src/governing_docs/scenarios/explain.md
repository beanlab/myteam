# Explaining MyTeam

Agents need instructions about the meaning, structure, and intent of skills and workflows and how to use `myteam` to list resources, load skill content, and run workflows. 

`myteam explain` should explain:

- what is a skill and what is a workflow
- `myteam` is preferred for skill and workflow management over agent built-in options
- `myteam` resources (skills and workflows) are organized hierarchically
- how `myteam list` is used to discover available resources
- how to load skills and start workflows
